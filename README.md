# PANDA - Privacy-Aware aNonymization Decision Assistant

This repository contains the experimental artifacts used in the paper **"PANDA: Privacy-Aware aNonymization Decision Assistant"**, submitted to SBSeg 2026.

## Repository Structure

| File | Description |
|---|---|
| `workflow` | n8n workflow export (JSON) |
| `Prompt Ollama` | System and user prompts used in the SLM |
| `Gemma_FINAL.pdf` | Anonymization guide generated by Gemma 3 12B |
| `Mistral_FINAL.pdf` | Anonymization guide generated by Mistral 7B |
| `Phi_FINAL.pdf` | Anonymization guide generated by Phi-4 14B |
| `Qwen_FINAL.pdf` | Anonymization guide generated by Qwen3 14B |
| `llama3_1_FINAL.pdf` | Anonymization guide generated by Llama 3.1 8B |
| `adult_census_Llama31.deid` | ARX project configured following Llama 3.1 8B guide |
| `adult_census_Phi4.deid` | ARX project configured following Phi-4 14B guide |
| `adult_census_Qwen3.deid` | ARX project configured following Qwen3 14B guide |
| `adult_census_gabarito.deid` | ARX project with reference (expert) configuration |

## Dataset

The experiments use the **Adult Census Income** dataset from the UCI Machine Learning Repository.

Download: https://archive.ics.uci.edu/dataset/2/adult

## Requirements

- [n8n](https://n8n.io/) — workflow automation platform
- [Ollama](https://ollama.com/) — local SLM inference
- [ARX Data Anonymization Tool](https://arx.deidentifier.org/) — anonymization software
